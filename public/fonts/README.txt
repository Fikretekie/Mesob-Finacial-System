PDF reports use custom fonts for non-Latin languages. Add the correct font file(s) here (public/fonts/) for the language(s) you use.

ARABIC (ar)
-----------
File name: Amiri-Regular.ttf
Download: https://fonts.google.com/specimen/Amiri (click "Download family")
Place: public/fonts/Amiri-Regular.ttf

AMHARIC / TIGRINYA (am, ti)
---------------------------
File name: NotoSansEthiopic-Regular.ttf
Download: https://fonts.google.com/noto/specimen/Noto+Sans+Ethiopic (click "Download family")
Place: public/fonts/NotoSansEthiopic-Regular.ttf

After adding a font, restart the dev server and generate the PDF again in that language.
